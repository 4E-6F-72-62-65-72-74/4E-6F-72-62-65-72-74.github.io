Specifikus szoba lekérése

Leírás

A megadott RID-jú szoba, és benne lakók lekérése. Több információt ad a lakosokról, mint amikor egyszerre több szoba kerül lekérdezésre

Kérés a szervertől

GET/api/rooms/{rid}

HEADERAuthorization = Bearer xxxxxxxxxx (Bearer tokened)

Válasz a szervertől

Sikeres bejelentkezés: (200 OK)

{
	"RID": 2,
	"Gender": 0,
	"Group": "L2",
	"Residents": [
		{
			"UID": 2,
			"BedNum": 1,
			"Name": "Kellia Bernardez",
			"Picture": null
		},
		...
	]
}

Helytelen RID (Room Identifier): (404 Not Found)

{
	"error": "There is no room with the specified ID!"
}

Hiányzik az Authorization fejléc: (400 Bad Request)

{
	"error": "Authorization header field not present!"
}

Hiányzik a Bearer szó az access token előtt: (400 Bad Request)

{
	"error": "Authorization isn't prefixed by \"Bearer\"."
}

Helytelen access token: (401 Unauthorized)

{
	"error": "Invalid access token!"
}

Helytelen URL: (404 Not Found)

{
	"error": "Page not found!"
}
